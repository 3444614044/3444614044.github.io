## 立即体验游戏

你可以：
1. **解压游戏**：
   ```bash
   tar -xzf git-escape-2026.05.03-09.27.tar.gz
   cd build
   ./git-escape.sh start
   ```

2. **启动HTTP服务器分享**：
   ```bash
   python3 -m http.server 8000 --directory local_distribution
   ```

3. **直接发送文件**：
   - 文件位置：/root/.openclaw/workspace/git-escape/git-escape-2026.05.03-09.27.tar.gz

## 游戏内容预览

**关卡1：文件操作迷宫**
